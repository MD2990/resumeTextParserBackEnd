
## Requirements
1- For development, you will only need Node.js v(14 and above) installed in your environnement.(https://nodejs.org/en)
2- Run npm install to install all node modules.
3- Run npm start to start the server you should see the server running on http://localhost:3001 in your terminal.

## Server Configuration
The server will be configured to run on port 3001.
this can be changed by changing the port number in the index.js file.

## Files Configuration
Rejected files will be saved in the rejected folder.
Accepted files will be saved in the accepted folder.
